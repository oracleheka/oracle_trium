Note de l’auteur

Heka est né dans la traversée de mes propres tempêtes.
Il n’est pas venu pour être vu.
Il est apparu, lentement, entre des silences, entre des nuits, entre des pertes.
Pas comme un oracle à tirer.
Mais comme un langage à écouter depuis l’intérieur.
Je garde pour moi ce que les cartes me disent.
Je les laisse parler sans bruit, sans spectacle, sans attente.
Je ne les retourne pas pour dire l’avenir.
Je les retourne pour comprendre le présent.
C’est là qu’est né Heka.
Dans l’absence de bruit. Dans la nécessité d’un langage juste.
Pas pour répondre.
Mais pour accompagner.
